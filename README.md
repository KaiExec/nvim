# Cheetsheets

## LSP

- Rename: <leader>rn (*Rename*)
- Goto Defination: gd (*go, defination*)
- Goto Declaration: gd (*go, declaration*)

# Configuration

## LSP

To add a LSP Server, you need:
1. MasonInstall(or Mason) to install the lsp server
2. Config it in *lsp.${serverName}.lua* (could copy from lsp-nvim)
3. Add your name of lsp Server in *lua.config.lsp* to enable it
